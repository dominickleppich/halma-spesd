Vorrausetzungen

Das Spiel läuft mit Java Runtime Enviroment 6 unter nahezu fast allen Systemen. Je schneller das System, desto weniger Zeit benötigt die KI für Spielzüge. Unter Systemen mit schwacher Grafikkarte besteht die Möglichkeit, dass Spielzüge relativ lange dauern.

Installation

Für die Installation unter Linux  gehen sie wie folgt vor:

  1. Laden sie das komplette HalmaPP - Team SPESD Paket herunter.
  2. Begeben sie sich im Terminal in das heruntergeladene Verzeichnis.
  3. Führen sie ant aus. Das Programm wird nun kompiliert. Dies kann bis zu wenige Minuten in Anspruch nehmen.

Start und Menüführung

Starten:
Starten sie das Spiel mit dem Befehl java -jar jar/HalmaPP.jar
über das Terminal. Das GUI sollte sich kurz darauf öffnen.
Mit java -jar jar/HalmaPP.jar --help kann eine Übersicht aller Parameter angezeigt werden.

Menüführung:

  • Datei:
    • Neu - Startet ein neues Spiel
    • Speichern - Speichert das aktuelle Spiel und alle Spielzüge in einer XML-Datei. (ist nur möglich, wenn das Spiel nicht über Netzwerk läuft)
    • Laden - Lädt eine XML-Spieldatei und lässt bis zu einem beliebigen Spielzug laden.
    • Beenden - Beendet das Spiel
  • Spieler:
    • Neuer Menschlicher Spieler
    • Neuer einfacher Computerspieler
    • Neuer mittlerer Computerspieler
    • Neuer schwerer Computerspieler
  • Hilfe:
    • Handbuch - Startet dieses Handbuch
    • Tutorial - Startet eine visuelle Einführung in das Spiel. Hier werden die Regeln kurz erklärt.
  • LAN
    • Netzwerkspiel hosten - Startet ein Netzwerkspiel. Hierbei ist der lokale Spieler rot und der Netzwerkspieler muss mit einem blauen Spieler dem Spiel beitreten.
    • Netzwerkspiel beitreten - Kann einem existierenden Netzwerkspiel beitreten. Es muss die IP-Adresse des Ziel-Computers eingegeben werden und im Anschluss ein blauer Spieler gewählt werden.

Spielzüge

Bei HalmaPP gibt es zwei Arten von Spielzügen:
Schieben - Beim Schieben wird die gewählte Spielfigur in eine der sechs möglichen Richtungen nur ein Feld verschoben, sofern dort keine andere Figur das Feld blockiert.
Springen - Beim Springen wird eine direkt neben der gewählten Figur stehende Figur (egal ob Feind oder Freund) benötigt. Linear von der gewählten Figur sollte sich hinter der angrenzenden Figur ein freies Feld befinden. Dann ist ein Sprung möglich. Sprünge lassen sich beliebig oft aneinander reihen, jedoch darf keine bereits besuchte Position erneut angesprungen werden. Nach erfolgreichem Springen ist keine normales Schieben in der gleichen Runde möglich.

Sieg

Der Sieg ist errungen, wenn das Zielhaus an der oberen Seite des Spielbretts in eigener Farbe gefüllt ist. Ob mit eigenen oder fremden Steinen spielt keine Rolle, solange alle 15 Felder besetzt sind. Sollte ein Spieler keinen Spielzug möglich haben, so ist die Partie als unentschieden zu werten.
